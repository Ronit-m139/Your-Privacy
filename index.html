Transparency

How We Handle Your Data

At Behind the algoritm, we believe that Transparency is essential when it comes to your data. This page explains what exactly is collected when you visit our website, why its colleceted, and also how its used.

01 // What is Google Analytics?

As this is a google sites, this websites uses google analytics to track user data. This data is used to help devlopers to improve their website and also the user experience.

Google analytics is a web service that provides statistics and some basic analyitical tools mainly for search engine optimisation (SEO) and also marketing purposes. The service is a part of googles marketing platform.

Its important to note that once google analyitics has already colleceted data about its users, its stored in a database where it cant be changed. This means users can't delete their data from google analyitics.

02 // Data collected when you visit this website

When any user visits Behind the Algorithm, the following data is collected through Google Analytics:

Automatically Collected Data

  • The day that the user views the website
  • The language the viewer uses the most
  • The country and town/city the user is viewing the website from
  • The platform the user is viewing the website on, including the browser and device model
  • The way the user found the website, for example entering a link, searching for the website, or following a link from another site. Google Analytics can also determine if the social media site has directed you through paid promotional content or genuine, 'organic' engagements
  • The interactions that the user has with the website. For example, the number of times the user has interacted with a component or the pages the user viewed on the website.
  • This data also includes events such as scrolling and whether or not the user is visiting the website for the first time
  • The number of times you have visited this website

What Google Analytics doesn't show: Googles analytics does not link your data that has been collected with your personal identity. All the data thats collected from you will remain anonymous. Although we aim to provide a comprehensive list of data tracked through Google Analytics, we highly suggest looking at Google's own documentation for more information.

03 // How we use this data

Our Commitments

We will never share data collected by Google Analytics with anyone outside of the Behind the Algorithm project team.

We use this data only to understand how visitors interact with the website and to improve the user experience for teenagers learning about data collection.

We do not collect personal information through forms, surveys or any other interactive features. This is an educational resource; not a platoform made to collect data

It's important to know that Google Analytics is one of many different web analytics services which also collect similar user data. The data collection methods demonstrated on this website are only for educational purposes.
